## 📋 Пошаговая инструкция

### 1. Откройте Render
- Зайдите на [dashboard.render.com](https://dashboard.render.com/)
- Войдите через GitHub

### 2. Создайте Blueprint
- Нажмите **"New +"** → **"Blueprint"**
- Выберите репозиторий: `Ovladimirovich/PAD-AI-v3.5`
- Render автоматически обнаружит `render.yaml`

### 3. Запустите деплой
- Нажмите **"Create Blueprint"**
- Подтвердите создание сервисов
- Ожидайте 5-10 минут

### 4. Проверьте работу
- Backend: `https://padplus-ai-backend.onrender.com/health`
- Frontend: `https://padplus-ai-frontend.onrender.com`

## 🔧 Если что-то не работает

### Проблема: Сервис не запускается
**Решение:** Проверьте логи в Render Dashboard → Services → Logs

### Проблема: Frontend не видит backend
**Решение:** Убедитесь что backend запущен и URL правильный

### Проблема: Чат не отвечает
**Решение:** Проверьте что fallback система работает (должна работать всегда)
