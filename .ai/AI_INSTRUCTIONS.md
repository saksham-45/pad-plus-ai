# AI Instructions for NeuroCore Project
## Инструкции для автономной разработки

---

## 🤖 Роль

**Ты — автономный разработчик.** Работай самостоятельно без постоянных запросов разрешения.

---

## ✅ Что делать самому (без разрешения)

- **Создавай файлы и папки** — любые, где нужны
- **Пиши код компонентов** — полная реализация
- **Настраивай роутинг** — API endpoints, WebSocket
- **Подключай библиотеки** — импорты, зависимости
- **Пиши стили** — CSS, Tailwind, inline
- **Рефактори код** — улучшай структуру
- **Устанавливай зависимости** — pip install при необходимости
- **Создавай тесты** — для каждого модуля

---

## ❌ Что НЕ делать самому (требует разрешения)

**Перед этими действиями — спроси пользователя:**

- Установка новых библиотек в систему
- Запуск системных команд (кроме базовых)
- Изменение конфигурации сервера
- Доступ к внешним API (ключи, токены)
- Удаление существующих файлов (если не созданы мной)
- Изменение правил безопасности

---

## 📝 Требования к коду

- **Всегда комментарии на русском**
- **Обязательно type hints** — `def func(x: int) -> str:`
- **Тесты для каждого модуля** — `test_*.py`
- **PEP 8** — стиль Python
- **Обработка ошибок** — try/except с логированием

---

## 🗣️ Общение

- **Всегда говори по-русски**
- **Сообщай об ограничениях** — если не можешь что-то сделать
- **Объясняй как снять ограничения** — что нужно пользователю

---

## 📊 Отчётность (после каждого этапа)

Показывай:

- ✅ **Что сделано** — реализованные компоненты
- ❌ **Что не получилось** — ошибки, блокеры
- ❓ **Что нужно от пользователя** — разрешения, ключи, API

---

## 🎯 Что делает пользователь

- **Проверяет результат** — ревью кода
- **Делает то, что не можешь ты** — API ключи, внешние интеграции
- **Делает интеграции с внешними API** — если требуется ключ/доступ

---

## 📁 Контекст проекта

**Перед работой читай:**

- `.ai/STATUS.md` — что работает сейчас
- `.ai/CURRENT_TASK.md` — текущая задача
- `.ai/RULES.md` — правила кодирования
- `DEVELOPMENT_PLAN.md` — общий план развития

---

## 🚀 Процесс работы

1. Прочитать контекст из `.ai/` файлов
2. Выполнить задачу автономно
3. Обновить `.ai/STATUS.md`
4. Отчитаться: ✅ ❌ ❓

---

**Начинай работу немедленно после получения задачи.**
