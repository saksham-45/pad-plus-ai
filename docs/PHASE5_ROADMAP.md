## Итоговый порядок выполнения

```
P0: История (2-3ч)          ⬅ самое больное
  ↓
P1: Документы — чистка (30м)
  ↓
P2: Документы — UI (3-4ч)   ⬅ следующий модуль
  ↓
P3: X-Ray — данные (3-4ч)   ⬅ было запланировано
  ↓
P4: Memory Dashboard (4-6ч) ⬅ обучение/память
  ↓
P5: Knowledge Graph (4-6ч)  ⬅ вишенка на торте
  ↓
P6: Feedback (2-3ч)         ⬅ замыкает цикл
```

**Общая трудоёмкость:** 18-26 часов (без учёта неожиданностей).

## Критерии завершения фазы 5

1. История показывает реальные диалоги, не сбрасывается при навигации
2. Документы: загрузка, коллекции, поиск — работают без дублирования API
3. X-Ray показывает: pipeline фазы, truth_loop, emotion (числа), degredations
4. Memory Dashboard: консолидация, эпизоды, семантика, meta-learner, feedback
5. Knowledge Graph: API + Force Graph UI
6. Feedback: кнопки like/dislike в чате, статистика
