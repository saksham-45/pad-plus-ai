# Impulse Architecture — место ядра в системе

## Зависимости

### Что зависит от импульса

| Компонент | Как зависит |
|-----------|-------------|
| **Генерация ответа (generate phase)** | `impulse_line` встраивается в системный промпт: `Твой импульс: "{question}"` |
| **Путь рассуждения** | Вектор импульса смещает априорные вероятности — модель отвечает в контексте текущего импульса |
| **Системный prompt** | `current_impulse.txt` синхронизируется с ядром и используется как часть контекста |

### Что НЕ зависит от импульса

| Компонент | Обоснование |
|-----------|-------------|
| **Эмоции** | Эмоциональный тон (tone, уверенность) управляется отдельно |
| **Память** | Эпизодическая, процедурная, семантическая память не зависит от импульса |
| **Персона** | Persona-контекст загружается независимо |
| **RAG** | Поиск по документам не фильтруется импульсом |
| **Провайдеры** | Выбор модели/провайдера не зависит от импульса |
| **X-Ray** | Система наблюдаемости ортогональна |
| **Auth** | Аутентификация не связана |

### Что может менять импульс

| Действие | Механизм |
|----------|----------|
| **PUT /api/v1/impulse** | Прямая установка весов через API |
| **PUT /api/v1/impulse/question** | Установка через текстовый запрос |
| **POST /api/v1/impulse/pop** | Восстановление предыдущего состояния |
| **Чат** | Косвенно — через вызовы API из фронтенда |

### Что не имеет права менять импульс

| Действие | Причина |
|----------|---------|
| **Эмоциональная система** | Импульс когнитивен, эмоции аффективны — разные слои |
| **Система памяти** | Память хранит факты, а не предпочтения |
| **X-Ray / метрики** | Наблюдение не меняет наблюдаемое |
| **Старт системы** | При старте восстанавливается последнее состояние (идемпотентность) |

## Технический долг

- **Дублирование роутов xray**: `frontend_router` имеет catch-all `/xray/{trace_id}`, который перекрывает специфичные пути `xray_router` (`/sessions`, `/trace/*`). Требуется унификация.
- **Нормализация весов**: не реализована (ARCH-012). Сознательное решение — веса как независимые интенсивности.
