# 🔄 Truth Loop — PAD+ AI v4.0

## Обзор

Truth Loop — система верификации утверждений в ответах.

**Файл:** `backend/core/truth_loop.py`

## Процесс

1. **Извлечение утверждений** — разбор ответа на атомарные claims
2. **Проверка** — поиск подтверждающих/опровергающих фактов
3. **Оценка уверенности** — расчёт confidence для каждого claim
4. **Выявление противоречий** — поиск конфликтующих фактов
5. **Реконструкция** — замена непроверенных утверждений на уточнения

## API

| Эндпоинт | Описание |
|----------|----------|
| `GET /api/v1/truth/stats` | Статистика верификации |
| `POST /api/v1/truth/verify` | Верификация текста |
