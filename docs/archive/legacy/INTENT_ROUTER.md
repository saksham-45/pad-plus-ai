# 🎯 Intent Router — PAD+ AI v4.0

## Обзор

Intent Router классифицирует намерения пользователя для выбора стратегии обработки.

**Файл:** `backend/core/intent_router.py`

## Категории намерений

| Категория | Описание | Пример |
|-----------|----------|--------|
| question | Вопросы | "Что такое Python?" |
| command | Команды | "Запусти рефлексию" |
| conversation | Беседа | "Привет, как дела?" |
| creative | Творчество | "Напиши стих" |
| reflective | Рефлексия | "Что ты думаешь о..." |
| learning | Обучение | "Научи меня..." |

## API

| Эндпоинт | Описание |
|----------|----------|
| `GET /api/v1/intent/stats` | Статистика намерений |
