# 🐛 Bugfix Report — 2026-04-08

## Исправленные ошибки

### 1. SmartCacheChroma: "Invalid isoformat string: ''"

**Проблема:**
```
SmartCacheChroma error: ValueError: Invalid isoformat string: ''
```

**Причина:**
В методе `store()` файла `backend/memory/smartcache_chroma.py` поле `created_at` сохранялось как пустая строка `""` когда `record.created_at` был равен `None`. Затем в методах `get()` и `search()` вызывался `datetime.fromisoformat("")` что вызывало ошибку.

**Исправление:**
```python
# Было:
"created_at": record.created_at or "",

# Стало:
"created_at": record.created_at or dt.now().isoformat(),
```

Также добавлена защита в методах `get()` и `search()`:
```python
created_at_str = metadata.get('created_at', '')
if not created_at_str:
    created_at_str = dt.now().isoformat()
created_at = dt.fromisoformat(created_at_str)
```

### 2. SmartCacheChroma: "MemoryRecord.__init__() got an unexpected keyword argument 'source'"

**Проблема:**
```
SmartCacheChroma error: TypeError: MemoryRecord.__init__() got an unexpected keyword argument 'source'
```

**Причина:**
В методах `get()` и `search()` создавались `MemoryRecord` с недопустимыми параметрами (`source`, `layer`, `confidence`, `depth`, `ttl`), которых нет в классе `MemoryRecord` из `base.py`.

**Исправление:**
```python
# Было:
record = MemoryRecord(
    id=record_id,
    text=text,
    source=metadata.get('source', 'user'),
    layer="husk",
    confidence=metadata.get('confidence', 0.5),
    depth=0,
    created_at=created_at,
    ttl=ttl
)

# Стало:
record = MemoryRecord(
    id=record_id,
    text=text,
    metadata={
        "source": metadata.get('source', 'user'),
        "layer": "husk",
        "confidence": metadata.get('confidence', 0.5),
        "ttl": ttl,
    },
    created_at=created_at_str
)
```

### 3. SmartCacheChroma: вызов несуществующего метода `is_expired()`

**Проблема:**
```
AttributeError: 'MemoryRecord' object has no attribute 'is_expired'
```

**Причина:**
В методе `get()` вызывался метод `record.is_expired()` которого нет в классе `MemoryRecord`.

**Исправление:**
```python
# Было:
if not record.is_expired():
    return record

# Стало:
ttl = record.metadata.get('ttl', self.DEFAULT_TTL)
created_at_str = record.created_at
if created_at_str:
    try:
        created_at = dt.fromisoformat(created_at_str)
        expires_at = created_at + timedelta(seconds=ttl)
        if datetime.now() <= expires_at:
            return record
    except (ValueError, TypeError):
        pass
```

### 4. Episodic save: "argument of type 'NoneType' is not iterable"

**Проблема:**
```
Episodic save error: TypeError: argument of type 'NoneType' is not iterable
```

**Причины:**
1. Переменная `episodic` могла быть не определена если в предыдущем блоке `try` происходила ошибка
2. Неправильная структура `emotion_before` - передавался `emotion_style` со структурой `{"tone": "...", ...}` вместо ожидаемой `{"уверенность": float, ...}`
3. Возможные `None` значения в параметрах

**Исправление:**

1. Инициализация `episodic = None` заранее:
```python
episodic = None  # Инициализируем заранее для избежания NameError
try:
    from memory.episodic import get_episodic_memory
    episodic = get_episodic_memory()
    ...
except Exception as e:
    logger.warning(f"Episodic memory error: {e}")
```

2. Преобразование структуры эмоций:
```python
emotion_before_dict = {
    "уверенность": float(emotion_state.get("уверенность", 0.5)),
    "удовольствие": float(emotion_state.get("удовольствие", 0.0)),
    "возбуждение": float(emotion_state.get("возбуждение", 0.0)),
}
```

3. Подготовка безопасных значений для всех параметров:
```python
safe_intent = intent if intent else "unknown"
safe_topic = safe_intent
safe_concepts = [safe_intent]
safe_user_id = user_id  # Может быть None для общих записей

if episodic is not None:
    try:
        episode = episodic.add_episode(
            user_message=str(user_message) if user_message else "",
            ai_response=str(result.response) if result.response else "",
            topic=safe_topic,
            intent=safe_intent,
            significance=min(significance, 1.0),
            emotion_before=emotion_before_dict,
            emotion_after={},
            concepts=safe_concepts,
            user_id=safe_user_id
        )
        result.episode_id = episode.id
    except Exception as ep_err:
        logger.warning(f"Episodic add_episode error: {type(ep_err).__name__}: {ep_err}")
else:
    logger.warning("Episodic memory not available, skipping save")
```

### 5. Type errors в PipelineResult.errors

**Проблема:**
`result.errors` имеет тип `List[str]`, но в код добавлялись словари.

**Исправление:**
```python
# Было:
result.errors.append({
    "module": "vector_memory",
    "error_type": type(e).__name__,
    "message": str(e)
})

# Стало:
result.errors.append(f"VectorMemory: {type(e).__name__}: {e}")
```
