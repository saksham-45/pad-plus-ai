# 😊 Эмоции PAD+ — PAD+ AI v4.0

## Обзор

Модель PAD+ расширяет классическую модель Pleasure-Arousal-Dominance тремя дополнительными измерениями.

## 6 измерений

| Измерение | Диапазон | Описание |
|-----------|----------|----------|
| Удовольствие | -1.0 ... +1.0 | Положительные/отрицательные чувства |
| Возбуждение | -1.0 ... +1.0 | Энергичность и активность |
| Доминирование | -1.0 ... +1.0 | Чувство контроля ситуации |
| Любопытство | 0.0 ... 1.0 | Интерес к новому |
| Уверенность | 0.0 ... 1.0 | Уверенность в ответах |
| Социальная связь | -1.0 ... +1.0 | Связь с пользователем |

**Файл:** `backend/emotion/pad_model.py`

## Затухание

- `DECAY_RATE = 0.001/сек`
- `DECAY_INTERVAL = 60 сек`
- Автоматический возврат к нейтральному состоянию

## Влияние на стиль

| Эмоция | Влияние на стиль |
|--------|-----------------|
| Pleasure > 0.3 | tone: friendly |
| Pleasure < -0.3 | tone: serious |
| Arousal > 0.3 | verbosity: detailed |
| Arousal < -0.3 | verbosity: concise |
| Confidence > 0.7 | color: confident |
| Confidence < 0.3 | color: uncertain |
