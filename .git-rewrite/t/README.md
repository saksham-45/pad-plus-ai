# PAD+ AI v3.5

**Когнитивный слой для искусственных интеллектуальных систем**

Самая продвинутая открытая реализация когнитивной архитектуры с полной наблюдаемостью внутренней работы системы.

## ✅ Особенности

✅ **Полная прозрачность работы** — вы видите ВСЁ что происходит внутри системы: каждую стадию обработки, каждую мысль, каждое решение, каждую проверку
✅ **Эмоциональная модель PAD** — система имеет внутреннее состояние: Удовольствие, Активация, Доминирование, Любопытство, Уверенность
✅ **Система X-Ray** — система полной наблюдаемости в реальном времени
✅ **Многомодельная архитектура** — работает с любым LLM: GPT-4, Gemini, Llama 3, Claude, любые локальные модели
✅ **Собственная память** — эпизодическая, семантическая, процедурная, фактическая
✅ **Цикл проверки правды TruthLoop**
✅ **Встроенная система безопасности**
✅ **Граф знаний**
✅ **Самостоятельное обучение и адаптация**

PAD+ это не просто обёртка над LLM. Это полноценный когнитивный слой который думает, запоминает, сомневается, решает и учится. LLM это всего лишь инструмент который он использует.

## 🛠️ Локальная установка

```bash
# Клонировать репозиторий
git clone https://github.com/Ovladimirovich/PAD-AI-v3.5.git
cd PAD-AI-v3.5

# Установить бэкенд зависимости
pip install -r requirements.txt

# Установить фронтенд зависимости
cd frontend
npm install

# Скопировать .env.example в .env и заполнить переменные
cp .env.example .env

# Применить миграции базы данных
python scripts/apply_migrations.py

# Запустить бэкенд
python backend/main.py

# В другом терминале запустить фронтенд
cd frontend
npm run dev
```
