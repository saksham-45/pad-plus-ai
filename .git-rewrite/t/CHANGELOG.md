# 📝 CHANGELOG — PAD+ AI

## [4.0.0] — 2026-04-08

### Добавлено
- **X-Ray система** — трассировка каждого шага мышления AI (`/api/v1/xray/*`)
- **ResponseGuard** — контроль идентичности (замена "я языковая модель" → "PAD+ AI")
- **CognitiveBudget** — определение режима мышления (FAST/BALANCED/DEEP)
- **ModelRouter v2** — автовыбор модели по сложности запроса + Circuit Breaker
- **Supabase Auth** — регистрация, вход, JWT авторизация
- **Управление провайдерами через UI** — добавление, редактирование, удаление API ключей
- **Поддержка 20+ LLM провайдеров** через LiteLLM
- **GigaChat через официальный SDK** — прямая интеграция с OAuth
- **Model Selector** — выбор модели из dropdown в header
- **Кнопка «Использовать»** — быстрое переключение активного провайдера
- **Dropdown моделей** при редактировании провайдера

### Исправлено
- Сохранение выбранной модели в localStorage
- Передача `provider` в запросах к чату
- Обработка пустых ответов от сервера (frontend JSON parsing)
- Авто-выбор модели при загрузке страницы
- `clearCache is not defined` ошибка
- Обработка ошибок в `generate_stream()` для GigaChat
- `except Exception:` без логирования — добавлено логирование

### Удалено
- Устаревшие отчёты тестов (43 файла)
- Мёртвый код (`adapters/`, `async_pad_model.py`, `async_rag_optimizer.py`)
- Устаревшие файлы документации (EVOLUTION.md, GENERATOR.md, и др.)
